# Desafio Técnico Projedata

Este repositório contém a solução desenvolvida para o teste prático do processo seletivo da Projedata.

## Tecnologias

- **Java 21**
- **Eclipse IDE**
- **Git**

## Como executar o projeto

Ter o **JDK 21** (ou superior) instalado.

1. **Clone o repositório**

```bash
git clone https://github.com/guilhermeshibuya/teste-pratico-iniflex
```

2. **Importando no Eclipse:**

- Abra o Eclipse
- Vá em `File > Import...`
- Selecione `General > Existing Projects into Workspace`
- Selecione a pasta do projeto clonado e clique em `Finish`

3. **Execução via Terminal (Opcional):**
   Acesse a pasta src e execute:

```bash
javac io/github/guilhermeshibuya/testepraticoiniflex/main/Main.java
java io/github/guilhermeshibuya/testepraticoiniflex/main/Main.java
```
